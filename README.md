

## Running NEXT JS

### Install Packages
```bash
npm install
```

### Running App
```bash
npm run dev
```


## Running Backend GQL Server API

### Go to backend directory
```bash
cd backend
```

### Install packages
```bash
npm install
```

### Running API Server
```bash
npm start
```
